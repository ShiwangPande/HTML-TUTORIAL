★ Arrays and Objects:

Arrays are used to store multiple values in a single variable, while objects are used to store complex data structures. JavaScript has many built-in methods that can be used to manipulate arrays and objects.

  • ➤    Arrays are used to store multiple values in a single variable, while objects are used to store complex data structures. JavaScript has many built-in methods that can be used to manipulate arrays and objects.

    ★ Arrays

  • ➤    An array is a type of object in JavaScript that allows you to store and manipulate multiple values in a single variable.
  • ➤    You can create an array by using square brackets [] and separating the values with commas.
  • ➤    Arrays are zero-indexed, which means the first element is stored at index 0, the second element at index 1, and so on.
  • ➤    You can access and modify individual elements in an array using their index, e.g. myArray[0] = "new value".
  • ➤    Arrays have a number of built-in methods, such as push() to add elements to the end of the array, pop() to remove the last element, and sort() to sort the elements in the array.
  • ➤    You can also use loops (such as for or forEach) to iterate over the elements in an array and perform actions on each one.
  • ➤    An array is a special type of object in JavaScript that stores a collection of data in a specific order.
  • ➤    The data in an array can be of any type, including strings, numbers, booleans, and other arrays or objects.
  • ➤    Arrays are indexed, meaning that each element is assigned a unique index number starting from 0, which can be used to access and manipulate the elements.
  • ➤    You can create an array using array literal syntax, which uses square brackets and comma-separated values: let myArray = [1, 2, 3];
  • ➤    You can also create an array using the Array() constructor function: let myArray = new Array(1, 2, 3);
  • ➤    Arrays have many built-in methods, such as push(), pop(), shift(), and unshift(), which can be used to add, remove, or manipulate elements in the array.

    • ★ Objects:

  • ➤    An object is another type of data structure in JavaScript, which allows you to store collections of key-value pairs.
  • ➤    You can create an object by using curly braces {} and defining the key-value pairs, separated by colons.
  • ➤    You can access and modify individual values in an object using their keys, e.g. myObject["key"] = "new value".
  • ➤    Objects can also have methods, which are functions that can be called on the object, such as myObject.method().
  • ➤    You can use loops (such as for...in) to iterate over the keys in an object and perform actions on each one.
  • ➤    Objects can be used to represent complex data structures, such as nested objects or arrays.
  • ➤    An object is a collection of key-value pairs, where each key is a string (or symbol) that is used to access the corresponding value.
  • ➤    Objects can store data of any type, including strings, numbers, booleans, arrays, and even other objects.
  • ➤    You can create an object using object literal syntax, which uses curly braces and key-value pairs: let myObj = {name: "John", age: 30};
  • ➤    You can also create an object using the Object() constructor function: let myObj = new Object({name: "John", age: 30});
  • ➤    You can access object properties using dot notation (e.g. myObj.name) or bracket notation (e.g. myObj["name"]).
  • ➤    Objects have many built-in methods and properties, such as Object.keys(), Object.values(), and Object.entries(), which can be used to access or manipulate the object's properties.

  • ➤    Functions are one of the key building blocks of JavaScript. They are used to group a set of related statements together to perform a specific task. There are several types of functions in JavaScript, including:

    • 1․ Named Functions:

  • ➤    These are functions that are declared with a name and can be called using that name. They can be defined using the function keyword followed by the function name and parameter list.

    • 2․ Anonymous Functions:

  • ➤    These are functions that are not named and are often used as arguments to other functions or as the return value of a function. They can be defined using the function keyword without a name.

    • 3․ Arrow Functions:

  • ➤    These are a more concise way of writing functions and were introduced in ES6. They use the => syntax instead of the function keyword and can be used to write both named and anonymous functions.

    • 4․ Higher-Order Functions:

  • ➤    These are functions that take other functions as arguments or return functions as their return value. They are often used for tasks such as filtering or mapping arrays.

    • 5․ Constructor Functions:

  • ➤    These are functions that are used to create new objects. They are typically defined using the function keyword and the new operator is used to create new instances of the object.

    • 6․ Callback Functions:

  • ➤    These are functions that are passed as arguments to another function and are called when a certain event or condition occurs. They are commonly used in asynchronous programming.

    • 7․ Immediately Invoked Function Expressions (IIFEs):

  • ➤    These are functions that are defined and immediately called without being assigned to a variable. They are often used to create a local scope for variables and functions to avoid polluting the global scope.
