✪ CSS Typography

This topic could cover how to use CSS to control the typography of text on a webpage, including font families, font sizes, font weights, line-height, and text-decoration.

★ CSS typography

  • ➤    Font-family: This property sets the font of an element. You can specify multiple fonts, and the browser will use the first one it finds available.
  • ➤    Font-size: This property sets the size of the font. You can use different units, such as pixels (px), ems (em), or percentages (%).
  • ➤   Font-weight: This property sets the thickness or boldness of the font. You can use keywords such as "normal" or "bold", or numeric values from 100 to 900..
  • ➤    Font-style: This property sets the style of the font. You can use keywords such as "normal", "italic", or "oblique".
  • ➤    Text-align: This property sets the horizontal alignment of the text. You can use keywords such as "left", "center", or "right".
  • ➤    Text-decoration: This property sets the decoration of the text. You can use keywords such as "none", "underline", or "line-through".
  • ➤    Line-height: This property sets the height of a line of text. You can use different units, such as pixels (px), ems (em), or percentages (%)..
  • ➤    Letter-spacing: This property sets the spacing between characters. You can use different units, such as pixels (px), ems (em), or percentages (%).
  • ➤    Word-spacing: This property sets the spacing between words. You can use different units, such as pixels (px), ems (em), or percentages (%).
  • ➤    Text-transform: This property sets the capitalization of the text. You can use keywords such as "none", "uppercase", or "lowercase".

These are just some of the properties that can be used to style text in CSS. There are many more, and they can be combined in various ways to achieve different effects. By using these properties, you can create attractive and readable typography on your web pages.
